   ‚Ä¢ Verify IP address is correct
   ‚Ä¢ Check firewall settings
   ‚Ä¢ Try TEST_PACKET command
   ‚Ä¢ Verify port 8081 is not in use

   ‚Ä¢ Check browser console for errors

   ‚Ä¢ Check server console output
   ‚Ä¢ Run 'ipconfig' in command prompt
   ‚Ä¢ Look for IPv4 address
   ‚Ä¢ Use 192.168.x.x or 10.x.x.x

PROBLEM: Server won't start
SOLUTION:
   ‚Ä¢ Check Python is installed
   ‚Ä¢ Verify ports are not in use
   ‚Ä¢ Run as administrator
   ‚Ä¢ Check antivirus isn't blocking
